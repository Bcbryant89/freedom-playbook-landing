# The Freedom Playbook Landing Page

This is the official sales page for **The Freedom Playbook** by Brandon B.

## Pages
- `index.html`: Main product and services page
- `thank-you.html`: Displayed after PayPal payment to confirm receipt

## Deployment
1. Push files to GitHub repository
2. Enable GitHub Pages in Settings > Pages
3. Choose `main` branch and `/root`
4. Visit the published site via the URL GitHub provides
